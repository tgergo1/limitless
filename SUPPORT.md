# Support

## Where to ask for help

- Usage questions and bug reports: GitHub Issues.
- Security issues: GitHub private security advisory.

## What to include

- Compiler and version.
- Operating system.
- Exact command line.
- Minimal reproducer.
- Expected vs actual output.

## Response expectations

Best-effort maintainer response. Security reports follow `SECURITY.md` timelines.
