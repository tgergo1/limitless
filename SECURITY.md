# Security Policy

## Supported versions

Only the latest tagged release is supported for security fixes.

## Reporting a vulnerability

Please open a private security advisory on GitHub for this repository.

If private advisory tooling is unavailable, open an issue with minimal exploit detail and request a secure contact channel.

## Response targets

- Initial acknowledgement: within 7 days.
- Triage/update cadence: every 14 days until resolved.
- Public disclosure: after fix is available or mitigation is documented.
